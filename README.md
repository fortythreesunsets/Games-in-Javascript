## Project name: 2D side scroller game

Live Demo: https://codepen.io/fortythreesunsets/pen/ZYGZEWm

## Technologies used:
- HTML
- CSS
- Vanilla JavaScript

## Features:
- Sprite animation
- Parallax background
- Collision detection
- Score display
- Mobile support

## Controls:
- Arrow Up: Jump
- Arrow Left/Arrow Right: Movement
- Enter: Restart game
