## Project name: 2D side scroller game

Live Demo: https://codepen.io/fortythreesunsets/pen/ZYGZEWm
Art by https://bevouliin.com/

## Technologies used:
- HTML
- CSS
- Vanilla JavaScript

## Features:
- Sprite animation
- Parallax background
- Collision detection
- Score display
- Full screen support
- Mobile support

## PC Controls:
- Arrow Up: Jump
- Arrow Left/Arrow Right: Movement
- Enter: Restart game

## Mobile Controls:
- Swipe Up: Jump
- Swipe Left/Swipe Right: Movement
- Swipe Down: Restart game
